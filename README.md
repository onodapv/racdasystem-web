## らくだシステム株式会社 公式サイト

このプロジェクトは、らくだシステム株式会社（Rakudasystem Co., Ltd.）の公式WEBサイトです。

- プロジェクトフォルダ: `D:\Dropbox\MYWORK\CursorWorks\RSC-WEB2`
- 想定ドメイン: `https://rakudasystem.co.jp`

### 構成

- `index.html` : トップページ（会社概要・サービス・実績・お問い合わせ）
- `assets/css/style.css` : デザイン・レイアウト・3D/AI風のスタイル
- `assets/js/main.js` : 軽微なアニメーションやインタラクション

ビルドツールなどは使用せず、静的HTML/CSS/JSのみで構成されています。  
そのため、任意のWEBサーバーにアップロードするだけで公開できます。

### ローカル確認方法

1. 本フォルダを任意の場所に展開（またはそのまま）します。
2. `index.html` をブラウザで開きます。
   - Windowsの場合は `index.html` をダブルクリックするか、ブラウザにドラッグ＆ドロップしてください。
